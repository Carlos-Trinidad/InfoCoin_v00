# InfoCoin
InfoCoin is a final project for a Platzi Vuejs Basic Course. It uses Coincap API to check cryptocurrency values.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
